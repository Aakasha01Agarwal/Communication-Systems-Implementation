**Libraries**
1) numpy 	(pip install numpy)
2) matplotlib   (pip install matplotlib)
3) tkinter

**How to use?**

Execute the program and you will be be popped up with a GUI window that will ask
you for the inputs of the DSB-SC (Am, Ac, Fm, Fc) and you will be presented with 
all the three graphs (Carrier signal, Message signal, and the DSb-SC)
